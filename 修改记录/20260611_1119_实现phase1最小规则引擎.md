## 修改内容

- 实现不依赖 LLM 的 6 人 MVP 狼人杀规则引擎。
- 新增角色、阵营、阶段、事件、夜晚行动、投票结果、游戏状态等核心模型。
- 实现角色分配、夜晚行动校验、夜晚结算、白天投票、平票处理、出局处理和胜负判断。
- 实现 player-scoped `PlayerView`，确保村民不看到隐藏身份，狼人只知道狼队友，预言家只获得阵营结果。
- 新增 `WerewolfEngine`，集中管理所有 game state 变更。
- 新增 `FakeAgent` 和 `SimulationRunner`，支持不接 LLM 自动跑完整对局。
- 新增 CLI：`python -m app.core.cli --games 3 --seed 42`。
- 新增核心规则测试，覆盖角色分配、夜晚结算、投票、胜负、非法动作、信息隔离和 100 局模拟。
- 更新 README 和新增 Phase 1 验收文档。

## 风险与后续

- 当前规则是 MVP 简化版，不包含警长、猎人、守卫、白痴、狼王、遗言规则和复杂发言轮次。
- 当前 FakeAgent 只用于规则闭环验证，不代表真实 AI 行为质量。
- 当前胜负规则为“狼人全出局好人胜，狼人数大于等于好人数狼人胜”，后续可根据板子配置扩展。
- 下一步进入 Phase 2：接入文本 LLM Agent，但仍必须通过 schema 和 action validator 后才能进入 game-core。
