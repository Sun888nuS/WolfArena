# 修改记录：收敛为在线真实 Agent 产品路径

## 背景

用户要求删除无用代码文件，不再保留离线模拟，只使用真实 Agent 进行在线对话，代码应按真实产品所需保留。

## 变更

- 删除离线模拟代码：
  - `backend/app/core/fake_agent.py`
  - `backend/app/core/simulation.py`
  - `backend/app/core/cli.py`
- 删除 `pyproject.toml` 中的 `wolfarena-sim` CLI 入口。
- 移除 `LLM_FORCE_FALLBACK` 配置和本地 fallback 决策路径。
- `TextAgent` 现在只调用 OpenAI-compatible 在线 provider；LLM 调用、JSON 解析、schema 或合法性校验失败时抛出 `LlmProviderError`。
- `agents/validators.py` 改为严格校验在线 Agent 输出，不再生成本地替代决策。
- `POST /api/games/{game_id}/advance` 和 `POST /api/games/{game_id}/actions` 在在线 Agent 不可用时返回 `503`。
- 前端删除 `llm_fallback_count` 展示，只显示在线 Agent 状态。
- 更新 `.env.example`、`.env.production.example` 和 `README.md`，说明产品路径为在线-only。
- 测试移除离线 fallback/FakeAgent 用例，改用测试替身模拟在线 Agent 接口，避免测试访问外部网络。
