# 修改记录：新增 LLM API 本地测试脚本

## 修改时间

2026-06-11 14:06

## 修改范围

- `backend/scripts/test_llm_api.py`
- `backend/scripts/`

## 修改内容

- 新增本地模型 API 连通性测试脚本，复用项目现有 `.env` 配置读取方式。
- 支持通过 `--model` 临时覆盖模型名，方便逐个测试中转站支持的模型。
- 支持通过 `--structured` 测试 `response_format=json_schema`，用于确认是否兼容游戏内结构化 AI 决策调用。
- 输出 API Key 时只展示脱敏内容，避免泄露真实密钥。

## 验证方式

- 本次先完成脚本生成，尚未实际发起外部 API 请求。
- 外部 API 测试需要使用本地配置的真实中转站密钥执行脚本。

## 风险与后续

- 不同中转站对 `response_format=json_schema` 的兼容性不一致，如果基础测试成功但结构化测试失败，需要在 provider 层增加兼容降级策略。
- 如果模型名不可用，脚本会展示中转站返回的 HTTP 错误正文，用于调整 `LLM_MODEL`。
